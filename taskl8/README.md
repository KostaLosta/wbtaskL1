L1.8 — Установка бита в числе (Go)

Запуск

```bash
go run ./taskl8 -n <int64> -i <0..63> -to <0|1>
```

Примеры

```bash
# Установить бит с индексом 0 в 0: 5 (0101₂) → 4 (0100₂)
go run ./taskl8 -n 5 -i 0 -to 0

# Установить бит с индексом 1 в 1: 5 (0101₂) → 7 (0111₂)
go run ./taskl8 -n 5 -i 1 -to 1

```

Поведение

- Программа печатает только итоговое число (результат операции) на стандартный вывод.
- Значение флага `-to`: если равно 1 — бит устанавливается в 1; любое иное значение — в 0.
- Индекс бита предполагается в диапазоне [0,63]; явной валидации в текущей реализации нет.
- Коды возврата: 0 (успех). Обработка ошибок ввода не реализована.

Полезные формулы и «шпаргалка»

- Маска i-го бита: mask := int64(1) << i
- Установить бит в 1: n = n | mask
- Сбросить бит в 0: n = n &^ mask
- Инвертировать бит: n = n ^ mask
- Проверить бит: ((n >> i) & 1) == 1

Примечания

- В Go оператор `&^` — это AND NOT: «n AND (NOT mask)».
- При манипуляциях со старшими битами учитывайте знаковый разряд `int64` (бит 63).
